# 萌娃记录 - 微信小程序

## 功能特性

- 🍼 **喂奶记录**：滚动选择器（10-300ml，步长10），默认上次容量
- 💊 **补剂记录**：类型标签+自定义，剂量默认1，单位选择（片/粒/毫升）
- 💩 **便便记录**：形状/颜色/量，形状可自定义+删除角标（正常/偏干/偏稀不可删）
- ✏️ **编辑功能**：点击记录卡片编辑按钮，编辑后返回记录列表
- 🗑️ **删除确认**：点击删除按钮弹出确认弹窗
- 📊 **统计卡片**：记录列表顶部显示喂奶/便便/补剂统计

## 项目结构

```
baby-tracker-wxapp/
├── app.js              # 小程序入口
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── sitemap.json        # 站点地图
├── pages/
│   ├── index/          # 首页（三个入口按钮）
│   ├── milk/           # 喂奶记录
│   ├── supplement/     # 补剂记录
│   ├── poop/           # 便便记录
│   └── records/        # 记录列表+统计
└── images/             # 底部导航图标（需自行添加）
```

## 使用方法

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择 `baby-tracker-wxapp` 文件夹
4. 填写自己的 AppID（或选择测试号）
5. 点击「编译」即可预览

## 注意事项

- 底部导航栏需要添加图标文件到 `images/` 目录：
  - `home.png` / `home-active.png`
  - `records.png` / `records-active.png`
- 数据存储使用 `wx.setStorageSync`，完全本地存储
- 小程序与网页版数据不互通
